    目录设计§

    src
-- compiler  #编译相关
-- core 	#核心代码
-- platforms #不同平台的支持
-- server	#服务端渲染

--shared 	#共享代码

    compiler§

    把模板解析成AST语法树，AST语法树优化，代码生成

    编译可以借助webpack vue-loader在构建时做，也可以在运行时做（不推荐）

    codegen 	#代码生产器
directives 	#指令
parser 		#解析器
